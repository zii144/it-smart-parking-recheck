停車單稽查系統 － 示範 QR Code

在另一台裝置（例如筆電）開啟這個頁面，然後用手機上的稽查員 APP 對準下列任一張圖片掃描， 即可實際測試相機掃描功能（而不是只點選 APP 內建的示範按鈕）。每組代碼對應的判定情境與 backend/app/seed.py 的 QR_DEMO_CODES 完全一致。

這些圖片是用 scripts/generate-demo-qr.mjs（qrcode 套件）產生的真實 QR Code， 內容就是代碼本身的文字（例如 QR-A1001），掃描後會呼叫與示範按鈕相同的 POST /api/qr/scan API。若之後 seed.py 的示範代碼有異動，重新執行 npm run generate-demo-qr 即可重新產生對應圖片。
